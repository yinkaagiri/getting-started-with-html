Basics of Web and HTML

Web was invented in the 1990's and it basically holds a colleccon of HTML documents. Web process has five or more major components which includes user, computer and Browser application, Internet and Server
HTML is the heart of Web, it's a basic language in which informations are entered on the web. It consists of text content, markups, references to other documents and links to other pages.

Texts: what you can see
Markup: what it looks like
Refrences: images and videos
Links: shortcut to other pages

Tags: letters that are written in specific format to execute command in HTML. Tags can be inline or block depending on the function.
